第二十届全国大学生智能汽车竞赛 - 极速光电龙芯组

项目简介

本开源项目为第二十届全国大学生智能汽车竞赛“极速光电组-龙芯”组别全功能实现代码，由福州大学参赛队伍开发。代码经过实车调试验证，可在赛道条件下实现直线速度3m/s的竞赛级性能表现。

本项目可作为参赛队伍的参考实现，展示了完整的赛道元素处理、控制算法与速度决策系统。

核心性能

• ✅ 满足第二十届智能车竞赛龙芯组别全部完赛要求

• ✅ 校内实验室实测直线速度可达3m/s

• ✅ 完整赛道元素识别与处理

• ✅ 稳定可靠的闭环控制系统

技术方案

整体架构


图像采集 → 图像处理 → 赛道识别 → 控制决策 → 电机/舵机控制


核心算法模块

1. 基础巡线控制

• 差比和算法：实现高精度中线跟踪

• 自适应权重调节

• 动态参数优化

2. 赛道元素识别

• 最长中线法：主赛道边界识别

• 角点检测算法：特殊元素特征提取

• 多特征融合判决

3. 状态机管理

• 元素锁定状态机：确保元素处理连续性

• 状态迁移逻辑

• 异常恢复机制

4. 智能决策系统

• 基于元素的速度决策：动态速度规划

• 预瞄距离自适应

• 过弯速度优化


环境要求

• Ubuntu20.04 、VS Code、MObaXterm

• 高帧率USB摄像头（这里选用的是龙邱的）

• 智能车竞赛标准赛道

调参建议

基础巡线

1. 初始调试时降低最大速度
2. 逐步调整差比和算法的比例系数
3. 验证巡线稳定性后再提速

元素处理

1. 单独测试每个元素的识别算法
2. 调整元素锁定阈值
3. 优化状态迁移条件

速度决策

1. 从保守速度开始调试
2. 逐步提高直道速度
3. 优化过弯减速比例

性能优化技巧

1. 图像处理优化
   • 合理设置ROI（感兴趣区域）
   • 使用整数运算替代浮点运算
   • 优化图像采样率

2. 控制优化
   • 使用增量式PID
   • 加入前馈控制
   • 动态调整控制周期

3. 系统优化
   • 合理分配中断优先级

   • 优化内存使用

   • 减少不必要的计算

注意事项

⚠️ 重要提示
1. 实际性能受硬件、赛道、环境等因素影响
2. 建议在理解算法原理的基础上进行调参
3. 首次上电请在低速下测试安全性
4. 定期备份稳定的参数配置文件
